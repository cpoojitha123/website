IOT Based Air and Noise Pollution Monitoring System

This system uses an Arduino Uno, a PM2.5 GP2Y1010AU0F Dust Smoke Particle Sensor. MQ-135 gas sensor is used to detect alcohol, benzene, carbon dioxide and MQ-9 detects carbon monoxide and methane gases present in the atmosphere. Ada fruit electret microphone can be used to measure the sound level in decibels which can be displayed on a 0.96 inch SPI OLED for reference. Whenever the particle count and the concentration of the gases (in PPM) reach a dangerous level, a buzzer gets activated. On the other hand, when the sound levels go beyond a certain point an LED is turned ON. A Wi-Fi module is used to push the data onto the cloud which can be later used for live tracking on a mobile phone.
